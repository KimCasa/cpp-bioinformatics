# Hydrophobic Protein Analysis (C++)

## Overview

This project analyzes protein sequences stored in FASTA format and identifies the proteins with the highest percentage of hydrophobic amino acids.

The program was developed as part of a bioinformatics programming course and demonstrates sequence parsing, biological data analysis, sorting algorithms, and file processing in C++.

## Biological Background

Hydrophobic amino acids play an important role in protein structure and function. Proteins enriched in hydrophobic residues are often associated with cell membranes and transmembrane domains.

This program calculates the percentage of hydrophobic amino acids in each protein sequence and reports the top-ranking proteins.

## Features

- Reads protein sequences from FASTA files
- Calculates protein length
- Computes hydrophobic amino acid content
- Filters out proteins shorter than 100 amino acids
- Identifies the top 15 proteins with the highest hydrophobic content
- Outputs protein descriptions, lengths, and hydrophobic percentages
